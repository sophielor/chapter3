CHAPTER 3

HTML: Headings


HTML has six header tags (< h1 > to < h6 >) that can be used to indicate headings of various sizes and thicknesses. Heading 1 is the largest and thickest, while heading 6 is the smallest and thinnest, and paragraphs fall below that level. These tags are used to describe the topic they precede and support all global attributes. This topic covers the appropriate use of these tags.
